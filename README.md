🔐 Pin Matching Solution

A simple and interactive PIN matching system built using HTML, CSS, and JavaScript. This project simulates a secure PIN verification interface where users can generate a PIN and match it using a keypad.

🚀 Live Demo

👉 (https://niloypaul07.github.io/Payoo-mobile-bank/)

📌 Features
🔢 Random 4-digit PIN generator
⌨️ On-screen numeric keypad
❌ Clear and backspace functionality
✅ PIN verification system
🎯 Success & error message display
🔒 Basic security simulation
📱 Responsive design
🛠️ Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)
